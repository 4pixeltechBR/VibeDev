# Release v3.4.1 — Modo País Reorientado para Devs Nativos

🌍 **Read in / Leia em:** [English](#-english) | [Português](#-português)

---

## 🇺🇸 English

**Release date:** 2026-07-11
**Scope:** VibeDev core (v1.5.1)
**Type:** Patch — reorientation, no behavior change.

### What changed

In v3.4.0 we marked China, Russia, and Ukraine as "Experimental" with the framing **"this framework is for enterprises operating in those markets"**. The author of VibeDev clarified the actual intent: **the framework is civic technology for native developers building tools for their own country**, using local infrastructure (servers, gateways, regulations, language).

This patch reorients the entire `references/modo-pais.md` to that model:

- **🇨🇳 CN, 🇷🇺 RU, 🇺🇦 UA** reclassified from "Experimental" → "Stable (community-maintained)".
- All 3 country blocks now include a section **"Lacunas conhecidas — contribuidor local bem-vindo"** (Known gaps — local contributor welcome) listing specific points where only someone living there can harden the content (current law status, regional DCs, gateway SDK versions, conflict-operation patterns).
- Disclaimer rewritten: now makes clear the file is a starting point for **native developers building for their own country**, NOT a guide for foreign companies entering other markets.
- "Quick decision matrix" expanded: each country now has its own recommended stack row instead of hypothesis cases.
- New section "Contributing" at the end: how to fork + PR + which countries most need local maintainers.

### Backward compatibility

- Pure framing change. No technical behavior changes in how `modo-pais.md` is loaded or applied.
- Existing projects (any country) continue to work identically.
- BR / US / EU users see no difference at all.

### File change footprint

```
vibedev/references/modo-pais.md     (modified — disclaimer, all 12 country blocks, ending)
vibedev/CHANGELOG.md                (modified — +14 lines for v1.5.1)
CHANGELOG.md                        (modified — +10 lines for v3.4.1)
docs/releases/v3.4.1.md             (new — this file)
```

### A note for native devs in CN / RU / UA

If you live in China, Russia, or Ukraine, work with dev communities there, and want to help make VibeDev more useful to your fellow native developers — your PRs are especially welcome. Each of those 3 country blocks ends with a list of specific gaps where only someone with current local context can fill them in:

- Current law / regulation status
- Gateway SDK versions in practice (not just docs)
- Local / regional hosting options
- Conflict-operation patterns (UA specifically)
- Sanctions / cross-border compliance status

Open a PR with your contribution. The framework grows because of contributors like you.
