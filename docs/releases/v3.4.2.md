## 🇺🇸 English

**Release date:** 2026-07-11
**Scope:** VibeDev core (v1.5.2)
**Type:** Patch — documentation only.

### What changed

This patch adds formal credit tracking and clickable links to the academic work that influenced VibeDev.

**Added:**
- `CREDITS.md` at repository root. Formal, dated, traceable record of people, papers, projects and communities that materially influenced VibeDev. Includes instructions on how to add a new entry via PR. Cross-linked from `CONTRIBUTING.md`.
- Hyperlinks in release notes (`v3.1.0`, `v3.2.0`) and `vibedev/CHANGELOG.md` pointing to:
  - Sandeco Macedo's GitHub profile: https://github.com/sandeco
  - His preprint: https://arxiv.org/abs/2607.00038

**Why v3.4.2 is a separate release:**
- v3.4.0 added the citation text
- v3.4.1 reclassified CN/RU/UA
- v3.4.2 makes the citation clickable and aggregated into a permanent, separately-maintained `CREDITS.md`

The progression preserves fine-grained traceability — if Sandeco's work becomes controversial or his position changes, we can version each adjustment.

```

vibedev/CHANGELOG.md                (modified — hyperlinked Macedo + arXiv + entry for v1.5.2)
CHANGELOG.md                        (modified — entry for v3.4.2)
docs/releases/v3.4.2.md             (new, this file)
```

### A note to Sandeco Macedo

If you're reading this: thank you. Your paper's epistemic discipline ended up shaping how VibeDev validates user work, names anti-patterns, and refuses to overstate what any single tool (prompt, context, harness, or loop) can do.

The citation lives in:
- CREDITS.md (permanent, version-controlled)
- v3.1.0 release notes (initial credit)
- v3.4.0 release notes (cross-influence map)
- v3.4.1 release notes (reorientation of country mode, inspired by epistemic discipline)
- v3.4.2 release notes (this formal aggregation)

PRs welcome if you want to refine any of the credit text or add your own contributions.
